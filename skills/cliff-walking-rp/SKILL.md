# cliff-walking-rp

Use this skill for changes in the `cliff_walking_rp` repository.

## Best Fit Tasks

- Update the Next.js viewer or its styling
- Adjust the Python training/export workflow
- Add or refresh repo guidance files
- Review run artifacts and site presentation

## Repository Notes

- The site is a static Next.js app under `site/`.
- Viewer styling is centralized in `site/src/app/globals.css`.
- Exported run bundles live in `site/public/runs/`.
- Keep edits narrow and preserve existing training outputs unless the user asks otherwise.

## Visual Default

- Use the repo's neo-minimal pastel green direction.
- Favor soft mint surfaces, muted borders, and accessible text contrast.
- Avoid heavy neon, purple-heavy palettes, or glossy dark-mode styling unless requested.
